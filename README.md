## How to Use

### With Coolify Enhanced (Recommended)

1. In Coolify, go to **Settings > Templates**
2. Click **+ Add Source**
3. Enter repository URL: `https://github.com/corelix-io/coolify-enhanced-templates`
4. Set folder path: `templates/compose`
5. Click **Save & Sync**
6. Go to **New Resource** — templates appear in the Services grid with a source label

### Manual (Any Coolify)

1. Click on a template's **Link to .yaml** link and copy the content
2. Open your **Coolify Dashboard**
3. Go to **Projects** and select or create a project
4. Click **+ Add New Resource**
5. Choose **Docker Compose Empty**
6. Paste the YAML content and click **Save**
7. Configure environment variables if needed
8. Click **Deploy**
